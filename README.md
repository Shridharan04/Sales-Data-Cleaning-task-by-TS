# Sales Data Cleaning

Performed data cleaning on Superstore Sales Dataset using Python and Pandas.

## Steps Performed
- Inspected data using head(), info(), shape
- Handled 11 missing values in Postal Code column
- Converted Order Date and Ship Date from text to datetime
- Created Shipping Days column
- Dropped unnecessary Row ID column
- Saved cleaned dataset as train_cleaned.csv

## Tools Used
- Python
- Pandas
- Jupyter Notebook
